MDN Responsive Design Mode

MDN Responsive Design Mode is a developer tool used to adapt websites to look and function properly on different devices. The developer tool allows websites to adapt appropriately to different screen sizes without sacrificing important functionality within the website. It does this by adjusting the content of the website to the proper screen size, device pixel ratio, and touch event simulation of the device in which the site is being accessed. MDN Responsive Design Mode can be toggled in the web-developer sub-menu of the Mozilla Firefox menu while using Mozilla's web development platform. Once in this design mode, developers can continue to work on their sites while knowing exactly how the site will be viewed on that specific type of device. Having a website that is responsive in design is crucial in Search Engine Optimization tactics to improve website rankings.



Other Developer Tools (DevTools) that Mozilla offers are:


  • Page Inspector
    Allows the user to view and edit different aspects of the page content and layout at once.
  • JavaScript Debugger
    Allows the user to walk through javascript code step by step to debug and pin-point issues.
  • Web Console
    Allows the user to access a command line interpreter to execute javascript expressions within the context of the page.
